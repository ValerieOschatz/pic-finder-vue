# pic-finder-vue

- приложение для поиска и просмотра фотографий, подключенное к API Unsplash.  
Реализован поиск фотографий по введенному слову, просмотр фотографий с описанием, также просмотр случайного фото.


### Стек

* JS
* HTML
* CSS


### Технологии

* Vue 3
* Vue Router
* SASS
* API
* Flexbox
* Адаптив, респонсив
* БЭМ Nested
